## 测试环境
- 操作系统：Windows
- Node.js 版本：>=18.x
- 浏览器：Chrome / Edge / Firefox 最新版本
- 测试数据：使用 store.ts 中的 mock 数据（3个批次，日期分布在2024年1月、2月、3月）
